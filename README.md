Este repositório contém o desenvolvimento de uma aplicação de contador utilizando a linguagem Dart e o framework Flutter. O projeto serviu como base para a compreensão do fluxo de estados, manipulação de widgets e a estrutura fundamental de um aplicativo mobile moderno e reativo.

---

### O que foi explorado:
* Widgets (Stateless e Stateful): Diferenciação entre componentes estáticos e aqueles que mantêm estados dinâmicos.
* Gerenciamento de Estado: Uso do método setState para atualizar a interface em tempo real.
* Estrutura de Projeto: Organização da função main, runApp e a árvore de widgets.
* Design de Interface: Utilização do Material Design para criação de botões, textos e alinhamentos.
* Tipagem Forte: Aplicação dos conceitos de segurança e tipagem da linguagem Dart.
